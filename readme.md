# Green運用代行サービス - ランディングページ

## プロジェクト概要

IT/Web業界特化のGreen媒体運用代行サービスのランディングページです。業界初の「面談課金モデル（1面談=2万円）」を採用し、初期費用・固定費ゼロの完全成功報酬型サービスを提供しています。

## 🎯 サービスの特徴

- **業界初の面談課金モデル**: 1面談あたり20,000円（税別）の明確な料金体系
- **完全リスクフリー**: 初期費用・月額固定費ゼロ
- **契約期間の縛りなし**: いつでも解約可能
- **IT/Web業界特化**: Green媒体に最適化された運用ノウハウ
- **一気通貫サポート**: 求人作成から面接調整まで完全代行
